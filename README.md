# Quote-ation 🌟

**Quote-ation** is a modern Android application designed to inspire you daily. It fetches random, high-quality quotes from various authors and presents them in a beautiful, animated interface.

## ✨ Features
- **Instant Inspiration**: Fetch new quotes with a single tap.
- **Beautiful UI**: Smooth fade-in animations and Material 3 design.
- **Daily Notifications**: Receive a handpicked quote every day to start your morning right.
- **Home Screen Widget**: Keep inspiration at your fingertips without even opening the app.
- **Share & Copy**: Easily share your favorite quotes with friends or copy them to your clipboard.

## 🛠️ Tech Stack
- **Language**: Java / Android SDK
- **Architecture**: Modern Android with WorkManager and AppWidgets
- **Networking**: OkHttp for fast and reliable API calls
- **UI Components**: Material Design 3

## 🚀 How to Run
1. Clone the repository.
2. Open in Android Studio (Koala or newer recommended).
3. Build and Run on your device or emulator.
